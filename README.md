# knowledge

Personal Obsidian-compatible markdown vault.

The notes live in `notes/` and remain openable in Obsidian. Publishing is
handled by GitHub Actions through the reusable workflow in
[`yoichiojima-2/nonnative`](https://github.com/yoichiojima-2/nonnative).

Enable **Settings -> Pages -> Source: GitHub Actions** in this repo before the
first deploy.
